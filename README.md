# Orca Experimentations

ORCA (https://github.com/hundredrabbits/Orca) is a live programming language in which I explore several musical ideas.
Here are the results.

- **chords.orca**
construct chords and chords progressions based on 12-tone interval system.
Midi drums and synths are used with Ableton (linked project comming soon)
